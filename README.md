# Next.js 13.2-Experimental Landing Page

## Features

- Next.js 13.2-Experimental w/ App directory
- TailwindCSS
- TypeScript
- Framer Motion
- API Routes

## Getting Started

First, install the dependencies:

```bash
pnpm i
```

Second, run the development server:

```bash
pnpm run dev
```
